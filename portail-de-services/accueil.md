# Accueil

## Accéder aux Widgets

Depuis l'**Accueil**, vous accédez à l'ensemble de vos **Widgets**.

Les Widgets sont directement utilisables depuis votre accueil sans nécessiter d'action supplémentaire.

## Ouvrir un module depuis un Widget

{% stepper %}
{% step %}
### Ouvrir le module

Cliquez sur **Ouvrir le module** depuis le Widget concerné pour être redirigé directement vers le module complet.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
✍️ Les Widgets permettent une utilisation rapide des fonctionnalités sans quitter l'accueil. Utilisez **Ouvrir le module** uniquement si vous avez besoin d'accéder à l'ensemble des fonctionnalités avancées.
{% endhint %}
