# Déposer un document

{% hint style="info" %}
✍️ **Prérequis** : L'utilisateur doit avoir activé son compte pour accéder au portail de service.
{% endhint %}

## Déposer un document

Deux méthodes sont disponibles pour déposer un document :

{% stepper %}
{% step %}
### Depuis le menu Documents

* Rendez-vous dans le menu **Documents**
* Cliquez sur **Déposer un document**
* Sélectionnez le fichier sur votre ordinateur
{% endstep %}

{% step %}
### Par glisser-déposer

* Depuis l'**Accueil**, sélectionnez le document sur votre ordinateur
* Glissez-le directement sur le Widget disponible
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
⚠️ Un classeur doit obligatoirement être sélectionné pour finaliser le dépôt.
{% endhint %}

{% stepper %}
{% step %}
### Finaliser le dépôt

* Rédigez une note si nécessaire (cette note sera visible sur la plateforme Ingeneo avec le document)
* Sélectionnez un **classeur**
* Cliquez sur **Enregistrer**
{% endstep %}
{% endstepper %}

{% hint style="success" %}
✅ Le document est déposé.
{% endhint %}

✍️ Vous pouvez visualiser le document à tout moment en cliquant dessus.
