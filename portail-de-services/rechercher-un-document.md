# Rechercher un document

**Rechercher et consulter un document**

**Rechercher un document**

Depuis l'**Accueil** ou le menu **Documents**, accédez à la barre de recherche.

La recherche fonctionne en alphanumérique et permet de retrouver un document par :

* Date
* Mot-clé
* Contenu présent directement dans le document

{% hint style="info" %}
✍️ Vous pouvez afficher de 10 à 100 éléments par page pour faciliter votre navigation.
{% endhint %}

**Consulter et agir sur un document**

{% stepper %}
{% step %}
### Ouvrir le document

Cliquez sur le document pour le visualiser.
{% endstep %}

{% step %}
### Actions disponibles

Plusieurs actions sont disponibles depuis la visualisation du document :

* **Annotations**
* **Impression**
* **Édition**
* **Téléchargement**
{% endstep %}
{% endstepper %}

{% hint style="info" %}
✍️ Sélectionnez l'action correspondant à votre besoin directement depuis la visualisation du document.
{% endhint %}
